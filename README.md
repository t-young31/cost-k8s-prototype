# cost-k8s-prototype

## Setup
Get a token from https://www.kubecost.com/install.html

## Usage
Create a `.env` file from `.env.sample`

```bash
make
# Forward some ports, view some dashboards
# Clean up by destroying the cluster
make destroy
```
